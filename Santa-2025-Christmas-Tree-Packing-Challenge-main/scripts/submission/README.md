# scripts/submission

Arquivos relacionados a *submission* e portfólios de receitas.

## Recomendações

O fluxo principal do repo é único:

* `python -m santa_packing` (ou `santa-solve` quando instalado)

## O que fica aqui

* `portfolios/*.json`: coleções de receitas para `sweep_ensemble`.
* `portfolio_merge.py`: workflow **legado** (baseado em binários C++ antigos).
